### 知识体系架构全景（共识状态）

| 维度 | 模型 |
|------|------|
| **知识分层** | L1 元知识（全局）+ L2 固化知识（项目级）+ L3 动态知识（长尾） |
| **投递通道** | 通道一：磁盘写入（L1+L2 强制注入） + 通道二：MCP 查询（L3 按需拉取） |
| **录入方式** | MCP 对话式反向写入 + 代码库自动扫描冷启动 |
| **质量控制** | 个人沙盒 → 团队提案 → 中央大脑 + TL 审批 |
| **生命周期** | 健康评分（手动标注 + 自动衰减）+ 逻辑删除 + 双引擎 GC |
| **作用域** | Org → Project → Module → Branch（仅 L2/L3 有作用域，L1 全局通用） |
| **多端适配** | 内容标准化 + 文件路径适配器 + 手动同步（8 种 vibecoding 工具） |
| **同步机制** | L1+L2 手动同步（磁盘写入），L3 无需同步（MCP 实时查询） |
